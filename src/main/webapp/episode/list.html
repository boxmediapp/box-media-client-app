search Publish All Changes help_outline
This will pushes the changes to brightcove in a batch for all the records that are in the NEEDS_TO_PUBLISH_CHANGES status. Normally the changes will be automatically pushed to the brightcove whenever you update a record. However, when multiple records are changed in a batch or application think you need to review the changes, the records will be in the NEEDS_TO_PUBLISH_CHANGES status. In this case, you can either go to the individual record detail page to push the changes, or you can do the batch operation.
