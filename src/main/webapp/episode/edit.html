This will import image from the Brightcove. In order to do this, you have to first attach this episode to an media item in the brightcove. This can be done either by clicking on "Create Place holder" button and then "Start Transcode" or if your media item already exist in the Brightcove, you can copy paste the Brightcove media item id into the Brightcove ID field and click on "Save" button. Then you will be able to imported the generatd image in the Brightcove

Add/Modify Availability Windows
Start End
help_outline
Click on the button to go to the Availability Windows Editor to modify the availability windows. Note that this is a separate entities from the schedule events which are used for the same purposes. If you have not specified any availability window, the linear schedule events will be used to set the availability window in the brightcove. If you specify the availability windows here, the schedule events will not used. You can specify the multiple available windows. In this case, the most appropriate availability window will be selected based on the current time and will be pushed to the brightcove to set the availability window there. This is because brightcove only supports a single ailability window.
